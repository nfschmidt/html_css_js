En cuanto a esta magistral bebida...

Podemos destacar sus ingredientes imprescindibles:

  • Mate
  • Bombilla
  • Yerba
  • Azucar..si lo desea
  • Agua..caliente!..y natural para la preparacion

Ademas su perfecta preparacion:

  1. Tomamos el mate y le ponemos la yerba hasta la mitad
  2. Sacudimos el mate tapandolo para sacarle el polvo
  3. Corremos la yerba para incrustar la bombilla
  4. Le agregamos agua natural para que no se queme la yerba
  5. Aca si lo desea le agrega azucar
  6. Ahora si..agua caliente (sin hervir) y listo!
